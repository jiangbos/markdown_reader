# Keyboard Shortcuts

Every shortcut below can be remapped in **Settings** (the gear icon, or `⌘;`).

| Shortcut | Action |
| -------- | ------ |
| ⌘P | Quick open — jump to any note |
| ⌘E | Toggle reading view for the current tab |
| ⌘\ | Show / hide the sidebar |
| ⌘S | Save immediately (everything auto-saves anyway) |
| ⌘B | Toggle **bold** |
| ⌘I | Toggle *italic* |
| ⌘F | Find in the current note |
| ⌘Z / ⌘⇧Z | Undo / redo |
| ⌃⌥→ / ⌃⌥← | Next / previous tab |
| ⌃⌥⇧→ / ⌃⌥⇧← | Move the current tab right / left |
| ⌘⌥W | Close the current tab |
| ⌘; | Open settings |

## Mouse

| Gesture | Action |
| ------- | ------ |
| Click a note | Open in the current tab |
| Double-click a note | Open in a new tab |
| Middle-click a tab | Close it |
| Drag a tab along the tab bar | Reorder tabs |
| Drag a tab onto another pane's tab bar | Move it to that pane |
| Drag a tab to a pane's edge | Split — left/right for side-by-side, top/bottom for stacked |
| Drag the divider between panes | Resize the split |
| Right-click in the sidebar | New note, new folder, rename, delete |
| ⌘+click a link | Open it |
| Click a checkbox | Toggle the task |
| Click a rendered table | Edit its source |
| Drag the sidebar divider | Resize the sidebar (double-click to reset) |

## Split view

Any layout works: one note on the left with two stacked on the right, three columns, a grid — drag tabs where you want them. Closing a pane's last tab folds the pane away automatically, and your layout is remembered per project.

## Projects

Click the folder name at the top of the sidebar to switch between projects. Type the first letters of a project's name to jump to it, press ⏎ to switch — or **⌘⏎ / ⌘-click to open it in a new browser tab**, so different browser tabs can show different projects.
